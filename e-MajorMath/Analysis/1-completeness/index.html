The Axiom of Completeness

§1.3 완비성 공리
Axiom
위로 유계인 $\R$ 의 공집합이 아닌 부분집합은 반드시 상한(least upper bound)을 갖는다.

$\Q$ 와 $\R$ 을 구별하는 핵심 공리이다. $\Q$ 에는 빈틈(gap)이 있어서 이 공리가 성립하지 않는다 — 예를 들어 $\{r \in \Q : r^2 < 2\}$ 는 $\Q$ 안에서 상한을 갖지 않는다.

Definitions

상계·하계·상한·하한

Def 1.3.1 — 상계와 하계

어떤 $b \in \R$ 이 존재하여 모든 $a \in A$ 에 대해 $a \le b$ 이면, $A \subseteq \R$ 를 위로 유계(bounded above)라 한다.

이때의 $b$ 를 $A$ 의 상계(upper bound)라 한다.

마찬가지로, 어떤 $l \in \R$ 이 존재하여 모든 $a \in A$ 에 대해 $l \le a$ 이면 $A$ 는 아래로 유계(bounded below)라 한다.

이때의 $l$ 을 하계(lower bound)라 한다.

Def 1.3.2 — 상한 (Supremum)

실수 $s$ 가 집합 $A \subseteq \R$ 의 상한(least upper bound, supremum)이려면:

(i) $s$ 는 $A$ 의 상계이다.

(ii) $b$ 가 $A$ 의 임의의 상계이면 $s \le b$ 이다.

$s = \sup A$ 로 표기한다. 하한(infimum)은 대칭적으로 정의되며 $\inf A$ 로 표기한다.

상한의 유일성
$s_1, s_2$ 가 모두 $A$ 의 상한이면, 조건 (ii)에 의해 $s_1 \le s_2$ 이고 $s_2 \le s_1$ 이므로 $s_1 = s_2$.
Definition

Maximum vs Supremum

Def 1.3.4 — 최대·최소

$a_0 \in A$ 이고 모든 $a \in A$ 에 대해 $a_0 \ge a$ 이면 $a_0$ 은 $A$ 의 최댓값(maximum)이다. 최솟값(minimum)도 대칭적으로 정의한다.

최댓값은 반드시 집합의 원소이지만, 상한은 그럴 필요가 없다. 최댓값이 존재하면 그것이 곧 상한이다.

Example 1.3.5
$(0,2) = \{x \in \R : 0 < x < 2\}$ 와 $[0,2] = \{x \in \R : 0 \le x \le 2\}$ 모두 $\sup = 2$ 이다. 그러나 $(0,2)$ 는 최댓값이 없고, $[0,2]$ 는 최댓값 $2$ 를 갖는다.
Example

$A = \left\{\dfrac{1}{n} : n \in \N\right\}$

$A = \{1, \frac{1}{2}, \frac{1}{3}, \ldots\}$ 이다.

$\sup A = 1$

(i) 모든 $n \in \N$ 에 대해 $\frac{1}{n} \le 1$ 이므로 $1$ 은 상계.

(ii) $b$ 가 임의의 상계이면 $1 \in A$ 이므로 $1 \le b$.

$\inf A = 0$
$0$ 은 하계이고, $0$ 보다 큰 어떤 수도 충분히 큰 $n$ 에 대해 $\frac{1}{n}$ 보다 크므로 하계가 아니다. 단, $0 \notin A$ 이므로 최솟값은 존재하지 않는다.

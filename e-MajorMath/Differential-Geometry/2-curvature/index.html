How Much Does a Curve Curve?

Chapter 2 — §2.1 Curvature
Concept

곡선에 두 가지 스칼라 함수를 대응시킨다: 곡률(curvature)과 비틀림(torsion).

곡률은 곡선이 직선에서 얼마나 벗어나는지를, 비틀림은 곡선이 평면에서 얼마나 벗어나는지를 측정한다.

직선의 곡률은 0이고, 평면곡선의 비틀림은 0이다.

Definition
Def 2.1.1 — 곡률 (Curvature, 단위속력 곡선)

$\vg$ 가 매개변수 $t$ 에 대한 단위속력 곡선(unit-speed curve)이면, 점 $\vg(t)$ 에서의 곡률은

$$\kappa(t) = \|\vgdd(t)\|$$

로 정의한다.

직관: 테일러 전개 $\vg(t + \Delta t) = \vg(t) + \vgd(t)\Delta t + \frac{1}{2}\vgdd(t)(\Delta t)^2 + \cdots$ 에서 접선 방향으로부터의 이탈량이 $\frac{1}{2}\|\vgdd(t)\|(\Delta t)^2$ 이므로, $\|\vgdd\|$ 가 곧 '휘어지는 정도'이다.

원의 곡률

중심 $(x_0, y_0)$, 반지름 $R$ 인 원의 단위속력 매개변수화:

$$\vg(t) = \left(x_0 + R\cos\frac{t}{R},\; y_0 + R\sin\frac{t}{R}\right)$$

$\vgdd(t) = \left(-\frac{1}{R}\cos\frac{t}{R},\; -\frac{1}{R}\sin\frac{t}{R}\right)$ 이므로

$$\kappa = \|\vgdd\| = \frac{1}{R}$$

작은 원은 곡률이 크고, 큰 원은 곡률이 작다.

Proposition
Prop 2.1.2 — $\vg(t)$ 가 $\R^3$ 에서의 정칙곡선(regular curve)이면, 곡률은 $$\kappa = \frac{\|\vgdd \times \vgd\|}{\|\vgd\|^3}$$

단위속력이 아닌 임의의 매개변수로 곡률을 계산할 수 있는 실용적인 공식이다. $\vgd \ne \vo$ (정칙)이기만 하면 된다.

증명 보기

단위속력 매개변수를 $s$ 라 하면, 연쇄법칙으로

$$\vgd = \frac{d\vg}{dt} = \frac{d\vg}{ds}\frac{ds}{dt}$$

이므로

$$\kappa = \left\|\frac{d^2\vg}{ds^2}\right\| = \left\|\frac{\frac{ds}{dt}\frac{d^2\vg}{dt^2} - \frac{d^2 s}{dt^2}\frac{d\vg}{dt}}{(ds/dt)^3}\right\|$$

$(ds/dt)^2 = \|\vgd\|^2 = \vgd \cdot \vgd$ 를 미분하면 $(ds/dt)(d^2s/dt^2) = \vgd \cdot \vgdd$ 이고, 벡터 삼중곱 항등식 $\va \times (\vb \times \va) = (\va \cdot \va)\vb - (\va \cdot \vb)\va$ 을 적용하면

$$\kappa = \frac{\|\vgdd \times \vgd\|}{\|\vgd\|^3} \quad\square$$
Example

Example 2.1.3 — 원형 나선 (Circular Helix)

정의

$z$-축을 축으로 하는 원형 나선:

$$\vg(\theta) = (a\cos\theta,\; a\sin\theta,\; b\theta), \quad \theta \in \R$$

$|a|$ = 반지름(radius), $2\pi|b|$ = 피치(pitch, 한 바퀴당 높이 변화).

나선은 반지름 $|a|$ 인 원기둥 $x^2 + y^2 = a^2$ 위에 놓인다.

곡률 계산

$\vgd = (-a\sin\theta,\; a\cos\theta,\; b)$, $\;\|\vgd\| = \sqrt{a^2 + b^2}$

$\vgdd = (-a\cos\theta,\; -a\sin\theta,\; 0)$

Prop 2.1.2를 적용하면

$$\kappa = \frac{|a|}{a^2 + b^2}$$
극한 경우 확인

$b = 0$ (평면 원): $\kappa = |a|/a^2 = 1/|a|$ — 원의 곡률과 일치.

$a = 0$ (직선): $\kappa = 0$ — 직선의 곡률과 일치.

나선의 곡률은 상수이다.

Plane Curves

§2.2 부호 곡률과 회전각
Definition
부호 단위법선 (Signed Unit Normal)

$\R^2$ 에서 단위속력 곡선 $\vg(s)$ 의 접선벡터를 $\mathbf{t} = \vgd$ 라 하자.

$\mathbf{t}$ 를 반시계 방향으로 $\pi/2$ 회전한 단위벡터를 부호 단위법선 $\vn_s$ 라 한다.

부호 곡률 (Signed Curvature)

Prop 1.2.4에 의해 $\dot{\mathbf{t}} = \vgdd$ 는 $\mathbf{t}$ 에 수직이므로 $\vn_s$ 에 평행하다. 따라서 스칼라 $\kappa_s$ 가 존재하여

$$\vgdd = \kappa_s \vn_s$$

$\kappa_s$ 를 부호 곡률(signed curvature)이라 하며, 양수·음수·0 모두 가능하다.

$\|\vn_s\| = 1$ 이므로 $\kappa = \|\vgdd\| = |\kappa_s|$, 즉 곡률은 부호 곡률의 절댓값이다.

Definition
Def 2.2.2 — 회전각 (Turning Angle)

단위속력 곡선 $\vg$ 의 접선벡터 방향을 각도 $\varphi(s)$ 로 나타내면

$$\vgd(s) = (\cos\varphi(s),\;\sin\varphi(s))$$

$\varphi$ 는 $2\pi$ 의 정수배만큼의 자유도가 있다. Prop 2.2.1에 의해, 초기값 $\varphi(s_0)$ 를 정하면 매끄러운 $\varphi$ 가 유일하게 결정된다.

이 $\varphi$ 를 $\vg$ 의 회전각(turning angle)이라 한다.

Proposition
Prop 2.2.3 — 단위속력 평면곡선 $\vg(s)$ 의 회전각을 $\varphi(s)$ 라 하면, $$\kappa_s = \frac{d\varphi}{ds}$$

부호 곡률은 접선벡터가 회전하는 속도이다. $\kappa_s > 0$ 이면 $\mathbf{t}$ 가 반시계 방향으로, $\kappa_s < 0$ 이면 시계 방향으로 회전한다.

증명 보기

$\mathbf{t} = (\cos\varphi, \sin\varphi)$ 이므로

$$\dot{\mathbf{t}} = \dot\varphi(-\sin\varphi, \cos\varphi)$$

$\vn_s = (-\sin\varphi, \cos\varphi)$ 이므로 $\dot{\mathbf{t}} = \dot\varphi\,\vn_s = \kappa_s\,\vn_s$. 따라서 $\kappa_s = d\varphi/ds$. $\square$

Corollary
Cor 2.2.5 — 닫힌 평면곡선의 전체 부호 곡률(total signed curvature)은 $2\pi$ 의 정수배이다. $$\int_0^\ell \kappa_s(s)\,ds = 2\pi n, \quad n \in \mathbb{Z}$$
증명 보기

길이 $\ell$ 인 닫힌 곡선이면 $\vg(s+\ell) = \vg(s)$ 이므로 $\vgd(\ell) = \vgd(0)$.

Prop 2.2.3에 의해 $\int_0^\ell \kappa_s\,ds = \varphi(\ell) - \varphi(0)$.

$(\cos\varphi(\ell), \sin\varphi(\ell)) = (\cos\varphi(0), \sin\varphi(0))$ 이므로 $\varphi(\ell) - \varphi(0)$ 는 $2\pi$ 의 정수배이다. $\square$

Theorem
Thm 2.2.6 — 임의의 매끄러운 함수 $k : (\alpha, \beta) \to \R$ 에 대해, 부호 곡률이 $k$ 인 단위속력 곡선 $\vg : (\alpha, \beta) \to \R^2$ 가 존재한다.

또한 같은 부호 곡률을 갖는 두 곡선은 $\R^2$ 의 직접 등거리변환(회전 + 평행이동)으로 서로 옮길 수 있다.

평면곡선은 부호 곡률에 의해 (등거리변환을 제외하면) 완전히 결정된다.

증명 보기

존재: $s_0 \in (\alpha,\beta)$ 를 고정하고 $\varphi(s) = \int_{s_0}^s k(u)\,du$ 로 두면

$$\vg(s) = \left(\int_{s_0}^s \cos\varphi(t)\,dt,\; \int_{s_0}^s \sin\varphi(t)\,dt\right)$$

는 단위속력이고 $\kappa_s = d\varphi/ds = k$.

유일성: 같은 $k$ 를 갖는 다른 곡선 $\tilde\vg$ 의 회전각을 $\tilde\varphi$ 라 하면, $k = d\tilde\varphi/ds$ 이므로 $\tilde\varphi(s) = \varphi(s) + \theta$ (상수 차이). 적분하면 $\tilde\vg = T_\va \circ \rho_\theta(\vg)$, 즉 회전 $\rho_\theta$ 와 평행이동 $T_\va$ 의 합성이다. $\square$

Example

Example 2.2.7 — 상수 부호 곡률

정칙 평면곡선의 곡률 $\kappa$ 가 양의 상수이면, $\kappa_s = \pm\kappa$ 이고 중간값 정리에 의해 $\kappa_s$ 는 상수이다.

Thm 2.2.6에 의해 $\kappa_s$ 가 상수인 모든 곡선은 반지름 $1/|\kappa_s|$ 인 원의 일부이다.

Example

Example 2.2.8 — Cornu's Spiral

$\kappa_s(s) = s$ (부호 곡률이 호 길이에 비례)로 두면, Thm 2.2.6의 증명에 따라 $\varphi(s) = s^2/2$ 이고

$$\vg(s) = \left(\int_0^s \cos\frac{t^2}{2}\,dt,\; \int_0^s \sin\frac{t^2}{2}\,dt\right)$$

이 적분은 초등함수로 표현할 수 없으며, 프레넬 적분(Fresnel integrals)이라 불린다. 이 곡선을 코르뉘 나선(Cornu's Spiral)이라 한다.

Space Curves

§2.3 프레네-세레 틀과 비틀림
Definition
프레네-세레 틀 (Frenet–Serret Frame)

$\R^3$ 에서 단위속력 곡선 $\vg(s)$ 의 곡률이 $\kappa(s) \ne 0$ 일 때, 세 단위벡터를 다음과 같이 정의한다:

접선벡터(tangent): $\;\mathbf{t} = \vgd$

주법선벡터(principal normal): $\;\vn = \dfrac{1}{\kappa}\dot{\mathbf{t}}$

종법선벡터(binormal): $\;\vb = \mathbf{t} \times \vn$

$\{\mathbf{t}, \vn, \vb\}$ 는 $\R^3$ 의 오른손 정규직교기저(right-handed orthonormal basis)이다.

Definition
비틀림 (Torsion)

$\vb$ 를 미분하면 $\dot{\vb} = \dot{\mathbf{t}} \times \vn + \mathbf{t} \times \dot{\vn} = \kappa\vn \times \vn + \mathbf{t} \times \dot{\vn} = \mathbf{t} \times \dot{\vn}$.

$\dot{\vb}$ 는 $\mathbf{t}$ 와 $\vb$ 모두에 수직이므로 $\vn$ 에 평행하다. 따라서 스칼라 $\tau$ 가 존재하여

$$\dot{\vb} = -\tau\vn$$

$\tau$ 를 $\vg$ 의 비틀림(torsion)이라 한다. 곡률이 0이 아닌 모든 정칙곡선에 대해 잘 정의된다.

Proposition
Prop 2.3.1 — $\vg(t)$ 가 $\R^3$ 에서 곡률이 0이 아닌 정칙곡선이면, 비틀림은 $$\tau = \frac{(\vgd \times \vgdd) \cdot \dddot{\vg}}{\|\vgd \times \vgdd\|^2}$$

Prop 2.1.2의 곡률 공식과 마찬가지로, 임의의 매개변수로 비틀림을 계산할 수 있는 공식이다.

Example

Example 2.3.2 — 원형 나선의 비틀림

$\vg(\theta) = (a\cos\theta,\; a\sin\theta,\; b\theta)$ 에 대해

$\vgd \times \vgdd = (ab\sin\theta,\; -ab\cos\theta,\; -a^2)$

$\dddot{\vg} = (a\sin\theta,\; -a\cos\theta,\; 0)$

$(\vgd \times \vgdd) \cdot \dddot{\vg} = a^2 b$, $\;\|\vgd \times \vgdd\|^2 = a^2(a^2 + b^2)$

$$\tau = \frac{b}{a^2 + b^2}$$
극한 경우 확인

$b = 0$ (평면 원): $\tau = 0$ — 평면곡선이므로 비틀림 없음.

나선의 비틀림도 곡률과 마찬가지로 상수이다.

Proposition
Prop 2.3.3 — 곡률이 0이 아닌 정칙곡선 $\vg$ 에 대해, $\vg$ 의 상이 평면에 포함될 필요충분조건은 $\tau = 0$ 이다.
증명 보기

(⇒) $\vg$ 가 평면 $\mathbf{v} \cdot \mathbf{N} = d$ 에 포함되면, $\mathbf{t} \cdot \mathbf{N} = 0$ 이고 $\dot{\mathbf{t}} \cdot \mathbf{N} = 0$, 즉 $\kappa\vn \cdot \mathbf{N} = 0$.

$\kappa \ne 0$ 이므로 $\vn \cdot \mathbf{N} = 0$. 따라서 $\vb = \mathbf{t} \times \vn$ 은 $\mathbf{N}$ 에 평행하고 상수이다. $\dot{\vb} = \vo$ 이므로 $\tau = 0$.

(⇐) $\tau = 0$ 이면 $\dot{\vb} = \vo$, 즉 $\vb$ 는 상수벡터. $\frac{d}{ds}(\vg \cdot \vb) = \mathbf{t} \cdot \vb = 0$ 이므로 $\vg \cdot \vb = d$ (상수), 즉 $\vg$ 는 평면에 포함된다. $\square$

Theorem
Thm 2.3.4 — 프레네-세레 방정식 (Frenet–Serret Equations)

단위속력 곡선 $\vg$ 의 곡률이 0이 아니면, $$\dot{\mathbf{t}} = \kappa\vn$$ $$\dot{\vn} = -\kappa\mathbf{t} + \tau\vb$$ $$\dot{\vb} = -\tau\vn$$

행렬로 쓰면

$$\begin{pmatrix} \dot{\mathbf{t}} \\ \dot{\vn} \\ \dot{\vb} \end{pmatrix} = \begin{pmatrix} 0 & \kappa & 0 \\ -\kappa & 0 & \tau \\ 0 & -\tau & 0 \end{pmatrix} \begin{pmatrix} \mathbf{t} \\ \vn \\ \vb \end{pmatrix}$$

계수 행렬이 반대칭(skew-symmetric)임에 주목하라. 이는 $\{\mathbf{t}, \vn, \vb\}$ 가 정규직교기저로 유지되는 것과 동치이다.

Proposition
Prop 2.3.5 — $\R^3$ 에서 단위속력 곡선의 곡률이 상수이고 비틀림이 0이면, 그 곡선은 원(의 일부)이다.
증명 보기

$\tau = 0$ 이므로 Prop 2.3.3에 의해 $\vg$ 는 평면 위의 곡선이고, $\vb$ 는 상수이다.

$\frac{d}{ds}\left(\vg + \frac{1}{\kappa}\vn\right) = \mathbf{t} + \frac{1}{\kappa}\dot{\vn} = \mathbf{t} + \frac{1}{\kappa}(-\kappa\mathbf{t}) = \vo$

따라서 $\vg + \frac{1}{\kappa}\vn = \va$ (상수), 즉 $\|\vg - \va\| = \frac{1}{\kappa}$.

$\vg$ 는 중심 $\va$, 반지름 $1/\kappa$ 인 구 위에 있고, 평면과의 교선이므로 원이다. 더 나아가 $\vg$ 가 놓인 평면이 $\va$ 를 지나므로 대원(great circle)이다. $\square$

Theorem
Thm 2.3.6 — 공간곡선의 기본 정리

$k > 0$ 과 $t$ 가 매끄러운 함수이면, 곡률이 $k$ 이고 비틀림이 $t$ 인 단위속력 곡선 $\vg$ 가 $\R^3$ 에 존재한다.

같은 곡률과 비틀림을 갖는 두 곡선은 $\R^3$ 의 직접 등거리변환으로 서로 옮길 수 있다.

평면곡선은 부호 곡률만으로 결정되었지만(Thm 2.2.6), 공간곡선은 곡률과 비틀림 두 함수로 (등거리변환을 제외하면) 완전히 결정된다.

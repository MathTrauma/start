Curves in the Plane and in Space

Chapter 1 — §1.1 What is a curve?
Concept
Level Curve

함수 $f(x,y)$ 와 상수 $c$ 에 대해, $f(x,y) = c$ 를 만족하는 점들의 집합

$$\mathcal{C} = \{(x,y) \in \R^2 \mid f(x,y) = c\}$$

를 등위곡선(level curve)이라 한다.

$\R^3$ 에서는 두 방정식 $f_1(x,y,z) = c_1$, $f_2(x,y,z) = c_2$ 로 곡선을 정의할 수 있다.

Definition
Def 1.1.1 — 매개변수 곡선 (Parametrized Curve)

$\R^n$ 에서의 매개변수 곡선이란, $-\infty \le \alpha < \beta \le \infty$ 인 $\alpha, \beta$ 에 대해

$$\vg : (\alpha, \beta) \to \R^n$$

인 사상(map)이다.

등위곡선 $\mathcal{C}$ 에 상(image)이 포함되는 매개변수 곡선을 $\mathcal{C}$ 의 매개변수화(parametrization)라 한다.

Example 1.1.2 — 포물선

$y = x^2$ 의 매개변수화: $\vg(t) = (t, t^2)$, $(\alpha,\beta) = (-\infty, \infty)$.

$(t^3, t^6)$ 이나 $(2t, 4t^2)$ 도 가능하므로 매개변수화는 유일하지 않다.

Example 1.1.3 — 원

$x^2 + y^2 = 1$: $\vg(t) = (\cos t, \sin t)$.

$\vg(t) = (t, \sqrt{1-t^2})$ 는 상반원만의 매개변수화이다.

Example 1.1.4 — Astroid

$\vg(t) = (\cos^3 t, \sin^3 t)$ → 등위곡선 $x^{2/3} + y^{2/3} = 1$.

Definition
매끄러운 곡선 (Smooth Curve)

매개변수 곡선 $\vg$ 의 도함수 $d^n\vg / dt^n$ 이 모든 $n \ge 1$ 과 모든 $t \in (\alpha, \beta)$ 에 대해 존재하면 $\vg$ 를 매끄럽다(smooth)고 한다.

이는 각 성분함수 $\gamma_1, \gamma_2, \ldots, \gamma_n$ 이 모두 매끄러운 것과 동치이다.

이후 이 책에서 다루는 모든 매개변수 곡선은 매끄러운 것으로 가정한다.

Definition
Def 1.1.5 — 접선벡터 (Tangent Vector)

매개변수 곡선 $\vg$ 의 일계도함수 $\vgd(t)$ 를 점 $\vg(t)$ 에서의 접선벡터(tangent vector)라 한다.

이는 현(chord)의 방향벡터의 극한으로 이해할 수 있다:

$$\frac{\vg(t + \delta t) - \vg(t)}{\delta t} \;\xrightarrow{\delta t \to 0}\; \frac{d\vg}{dt}$$

$d\vg/dt \ne \vo$ 이면, 곡선 $\mathcal{C}$ 위의 점 $\mathbf{p}$ 에서 $\mathbf{p}$ 를 지나고 $d\vg/dt$ 에 평행한 직선을 접선(tangent line)이라 한다.

Proposition
Prop 1.1.6 — 매개변수 곡선의 접선벡터가 상수이면, 곡선의 상(image)은 직선(의 일부)이다.
증명 보기

$\vgd(t) = \va$ (상수벡터)이면, 성분별로 적분하여

$$\vg(t) = \int \frac{d\vg}{dt}\,dt = \int \va\,dt = t\,\va + \vb$$

여기서 $\vb$ 는 적분상수(상수벡터)이다.

$\va \ne \vo$ 이면 이것은 $\vb$ 를 지나고 $\va$ 에 평행한 직선의 매개변수 방정식이다. $\va = \vo$ 이면 $\vg$ 의 상은 점 $\vb$ 하나이다. $\square$

Example

Example 1.1.7 — Limaçon

$\vg(t) = \big((1 + 2\cos t)\cos t,\;(1 + 2\cos t)\sin t\big)$, $\;t \in \R$.

원점에서 자기교차(self-intersection)가 발생한다: $\vg(2\pi/3) = \vg(4\pi/3) = \vo$.

접선벡터
$$\vgd(t) = (-\sin t - 2\sin 2t,\;\cos t + 2\cos 2t)$$

$\vgd(2\pi/3) = (\sqrt{3}/2,\;-3/2)$

$\vgd(4\pi/3) = (-\sqrt{3}/2,\;-3/2)$

같은 점 $\vo$ 에서 접선벡터가 두 개 — 곡선의 '점'이란 무엇인가에 대한 주의가 필요하다.

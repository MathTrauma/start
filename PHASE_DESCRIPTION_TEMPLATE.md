# Phase 설명 템플릿

이 템플릿을 사용하여 코드 없이 각 phase를 설명할 수 있습니다.

## 기본 형식

```
PHASE [번호]: [단계 제목]
시간: [시작시간]-[종료시간]초

[구성 요소들]
```

## 구성 요소 표기법

### 1. 점 표시
```
점 [이름] 표시 @ [시간]초 (라벨 위치: [dx], [dy])
```

### 2. 선분 그리기
```
선분 [시작점][끝점] 그리기 @ [시작시간]초 (duration: [지속시간]초)
```

### 3. 삼각형 그리기
```
삼각형 [점1][점2][점3] 그리기 @ [시작시간]초 (duration: [지속시간]초)
```

### 4. 직각 표시
```
직각 [점1][꼭짓점][점2] 표시 @ [시간]초 (크기: [size])
```

### 5. 원 그리기
```
원 [중심] 그리기 @ [시작시간]초 (반지름: [r], duration: [지속시간]초)
```

### 6. 주석/설명
```
주석 "[텍스트]" @ 위치([x], [y]) @ [시간]초
```

## 완전한 예시

### 현재 프로젝트 (_5_test)

**초기 조건:**
- 점 B(-3, -3), C(3, -3)
- 삼각형 ABC는 이등변삼각형, 밑각 55°
- D = BC의 중점
- E = D를 AB에 투영한 점
- M = DE의 중점
- X = CE와 AM의 교점

---

**PHASE 1: 삼각형 ABC 그리기**
시간: 0-2초

- 점 A 표시 @ 0초 (라벨 위치: -10, -5)
- 점 B 표시 @ 0.2초 (라벨 위치: -10, 15)
- 점 C 표시 @ 0.4초 (라벨 위치: 5, 15)
- 삼각형 ABC 그리기 @ 0.6초 (duration: 1.4초)

---

**PHASE 2: 중점 D와 중선 AD**
시간: 2-4초

- 점 D 표시 @ 2.0초 (라벨 위치: 5, -5)
- 선분 AD 그리기 @ 2.3초 (duration: 1.0초)
- 직각 ADC 표시 @ 3.4초 (크기: 0.3)

---

**PHASE 3: 투영점 E와 선분 DE**
시간: 4-5초

- 선분 DE 그리기 @ 4.0초 (duration: 1.0초)
- 점 E 표시 @ 4.0초 (라벨 위치: -10, 5)
- 직각 DEB 표시 @ 5.0초 (크기: 0.3)

---

**PHASE 4: 중점 M과 보조선들**
시간: 5-7초

- 점 M 표시 @ 5.0초 (라벨 위치: -12, 12)
- 선분 CE 그리기 @ 5.3초 (duration: 1.0초)
- 선분 AM 그리기 @ 6.3초 (duration: 1.0초)

---

**PHASE 5: 교점 X**
시간: 7-8초

- 점 X 표시 @ 7.0초 (라벨 위치: 5, -5)

## 타이밍 규칙

1. **연속성**: 한 phase의 끝 시간 = 다음 phase의 시작 시간
2. **선분 → 직각**: 선분이 끝나는 시간 + 0.1~0.2초 후에 직각 표시
3. **점 → 선분**: 점 표시 후 0.1~0.3초 후에 선분 시작
4. **권장 duration**:
   - 점 페이드인: 0.3초
   - 짧은 선분: 0.5-1.0초
   - 긴 선분/삼각형: 1.0-2.0초
   - 직각 마커: 0.3초

## 사용 방법

1. 이 템플릿을 복사하여 새 파일 생성
2. "초기 조건"에 점들과 관계 정의
3. 각 PHASE 섹션 작성
4. 개발자에게 전달하면 코드로 구현

## 예시: 간단한 프로젝트

### 프로젝트명: 정삼각형의 무게중심

**초기 조건:**
- 점 A(0, 2), B(-1.732, -1), C(1.732, -1)
- G = 삼각형 ABC의 무게중심
- D, E, F = 각 변의 중점

---

**PHASE 1: 정삼각형 그리기**
시간: 0-2초

- 점 A 표시 @ 0초 (라벨 위치: 0, 10)
- 점 B 표시 @ 0.2초 (라벨 위치: -15, -5)
- 점 C 표시 @ 0.4초 (라벨 위치: 15, -5)
- 삼각형 ABC 그리기 @ 0.6초 (duration: 1.4초)

---

**PHASE 2: 중점들 표시**
시간: 2-4초

- 점 D 표시 @ 2.0초 (라벨 위치: -10, 5)
- 점 E 표시 @ 2.3초 (라벨 위치: 10, 5)
- 점 F 표시 @ 2.6초 (라벨 위치: 0, -10)

---

**PHASE 3: 중선 그리기**
시간: 4-7초

- 선분 AD 그리기 @ 4.0초 (duration: 1.0초)
- 선분 BE 그리기 @ 5.0초 (duration: 1.0초)
- 선분 CF 그리기 @ 6.0초 (duration: 1.0초)

---

**PHASE 4: 무게중심 표시**
시간: 7-8초

- 점 G 표시 @ 7.0초 (라벨 위치: 5, -5)

## 체크리스트

작성 시 확인사항:
- [ ] 모든 점에 라벨 위치가 지정되었는가?
- [ ] Phase 시간이 겹치지 않는가?
- [ ] 직각은 관련 선분이 끝난 후에 표시되는가?
- [ ] 애니메이션 순서가 논리적인가?
- [ ] duration 값이 적절한가?
- [ ] 초기 조건에 모든 점이 정의되었는가?
